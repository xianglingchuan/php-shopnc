数据格式约定

返回数据为json格式，code为200表示数据正常，datas为返回的数据 

{code: 200, datas: {}}

返回值

获取数据

//有内容直接返回数组，无内容返回空数组[]
//hasmore为是否有下一页
{"code":200,"hasmore":true,"datas":{"goods_list":[]}}

操作成功

//操作成返回"1"
{"code":200,"datas":{"1"}}

错误

//出现错误返回error，error内为错误信息
{"code":200,"datas":{"error":"参数错误"}}
//登录失败，login:0
{"code":200,"login":"0","datas":{"error":"\u8bf7\u767b\u9646"}}

煤炭列表

煤炭列表接口(get)

  index.php?act=goods&op=goods_list

请求参数

  • key 排序方式 1-销量 2-浏览量 3-价格 空-按最新发布排序
  • order 排序方式 1-升序 2-降序
  • page 每页数量
  • curpage 当前页码
  • gc_id 分类编号
  • keyword 搜索关键字
  • gc_id和keyword二选一不能同时出现

返回数据

  • goods_id 煤炭编号
  • goods_name 煤炭名称
  • goods_price 煤炭价格
  • goods_marketprice 煤炭市场价
  • goods_salenum 销量
  • evaluation_good_star 评价星级
  • evaluation_count 评价数
  • group_flag 是否团购
  • xianshi_flag 是否限时折扣
  • goods_image 图片名称
  • goods_image_url 图片地址
  • is_fcode 是否为F码煤炭 1-是 0-否
  • is_appoint 是否是预约煤炭 1-是 0-否
  • is_presell 是否是预售煤炭 1-是 0-否
  • have_gift 是否拥有赠品 1-是 0-否

购物车购买：第一个数字为购物车编号，第二个数字为购买数量，用竖线分割。多组用半角逗号分割，例：232|1,110|2 232煤炭购买1个，110煤炭购买2个

  • pay_sn 支付编号
